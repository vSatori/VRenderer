# VRenderer
VRenderer是一个简单的软光栅渲染器，用于图形学知识的实践。


![](https://github.com/vSatori/VRenderer/blob/main/screenshots/cubemap.png)


![](https://github.com/vSatori/VRenderer/blob/main/screenshots/bachong.png)

## 特性

- 左手坐标系
- 自由相机
- 齐次空间裁剪
- 正面/背面剔除
- 深度测试
- 纹理映射
- 阴影贴图
- 立方体贴图
- 基础光照模型
- Blinn-Phong着色
- Alpha混合
