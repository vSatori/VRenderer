# VRenderer
VRenderer is a simple soft rasterization renderer,for the practice of graphics knowledge.

## Features

- Homogeneous Space Clipping
- cull
- Depth Test
- Texture
- Shadow Mapping
- Cube Mapping
- Blinn-Phong Shading
- Alpha Blending
- MSAA


## Dependencies

- [stb_image](https://github.com/nothings/stb)
- [MMDFormats](https://github.com/oguna/MMDFormats)
